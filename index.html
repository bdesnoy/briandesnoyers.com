Algorithm to Maximize Employee Influence (Python)

This is a program that finds the optimum employees to influence in a company to spread propaganda within a company, as described in the problem description. The program is written in Python and was written as part of coursework for the Algorithms and Data Structures course at Northeastern University. A greedy algorithm was used to find the solution, as described in the Readme.pdf file. The list of employees was maintained in a priority queue implementation to ensure that it could process a list of ~800,000 employees in under one minute.
